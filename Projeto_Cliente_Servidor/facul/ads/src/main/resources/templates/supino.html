SUPINO RETO COM BARRA

MÚSCULOS TRABALHADOS:

1. Deite no banco;

2. Pegue a barra com distância maior do que a dos ombros;

3. Levante o peso;

4. Desça a barra e realize a flexão dos cotovelos para baixo;

5. Empurre a barra para cima concentrando a força nos músculos do peito;

6. Repita os movimentos.
